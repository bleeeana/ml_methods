# Методы машинного обучения

Этот репозиторий содержит Jupyter notebook (`methods.ipynb`) с реализацией различных методов машинного обучения.

## Содержание

В настоящее время в файле представлены следующие методы:

1. **Классификатор наивный Байес** - вероятностный классификатор на основе теоремы Байеса
2. **Линейная регрессия** - метод предсказания непрерывных значений
3. **Логистическая регрессия** - метод классификации с использованием сигмоиды
4. **Метод опорных векторов (SVM)** - алгоритм классификации 
5. **Решающие деревья** - алгоритмы классификации и регрессии на основе деревьев

## Статус проекта

⚠️ **Важно:** Файл `methods.ipynb` находится в разработке и еще не завершен. Он будет регулярно редактироваться и пополняться новыми методами машинного обучения.

## Использование

Для работы с notebook необходимо установить следующие зависимости:

```bash
pip install pandas numpy matplotlib scikit-learn jupyter
```

Затем запустите Jupyter Notebook:

```bash
jupyter notebook methods.ipynb
```

## Структура

Каждый метод в notebook содержит:
- Описание алгоритма
- Примеры кода с реализацией
- Визуализации результатов (где применимо)
- Примеры использования на данных
